# Российская школа психоанализа

Демонстрационный образовательный одностраничный сайт о российской школе психоанализа.

## Что реализовано

- одностраничная структура с навигацией по разделам;
- адаптивная верстка;
- фильтры программ;
- интерактивная хронология;
- поиск по библиотеке;
- словарь-аккордеон;
- модальные окна программ;
- самопроверка маршрута обучения;
- демонстрационная форма заявки;
- базовые SEO-файлы: `robots.txt`, `sitemap.xml`.

## Важно

Форма работает в демонстрационном режиме. Данные не отправляются специалисту и не сохраняются на сервере. Для реального проекта нужно подключить сервис форм и добавить политику обработки персональных данных.
