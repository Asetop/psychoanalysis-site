# Российская школа психоанализа

Статический сайт с формой заявки. На Netlify заявки принимаются через встроенный Netlify Forms.

Локальный сервер остается как дополнительный вариант:

- принимает `POST /api/apply`;
- проверяет обязательные поля;
- сохраняет заявку в `data/applications.jsonl`;
- отправляет письмо владельцу сайта через Resend, если настроены `OWNER_EMAIL`, `FROM_EMAIL` и `RESEND_API_KEY`.

## Запуск

1. Установите Node.js 20 или новее.
2. Скопируйте `.env.example` в `.env`.
3. Заполните `OWNER_EMAIL`, `FROM_EMAIL`, `RESEND_API_KEY`.
4. Запустите:

```bash
npm start
```

После запуска сайт будет доступен на `http://localhost:3000`.

## Публикация на Netlify

В проект добавлен `netlify.toml`. Папка публикации: корень этой папки (`outputs`).

Форма называется `application`; после публикации заявки будут видны в Netlify: Site configuration / Forms.

Почтовые уведомления можно подключить позже в Netlify Forms notifications или через Resend.

## Какие данные нужны для реальной отправки

- email владельца сайта;
- подтвержденный адрес отправителя на домене, который добавлен и verified в Resend;
- API-ключ Resend или другой почтовый провайдер, если захотите заменить адаптер отправки.
